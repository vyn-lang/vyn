# Control Flow

Control flow determines the order in which your program executes code. Vyn provides conditional statements to make decisions based on conditions.

## In This Section

- **[If Statements](if-statements.md)** - Making decisions in your code

_More control flow features coming soon: loops, match statements, and more._

## Quick Overview

```vyn
let age: Int = 18

if age >= 18 {
    stdout# "Adult"
} else {
    stdout# "Minor"
}
```
