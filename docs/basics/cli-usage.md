# CLI Usage

The Vyn command-line interface provides several commands for working with Vyn programs.

## Basic Usage

```bash
vyn [OPTIONS] <COMMAND>
```

## Commands

### `run` - Run a Program

Execute a Vyn source file:

```bash
vyn run program.vyn
```

This compiles and runs your program in one step.

### `check` - Type Check

Verify your program's types without running it:

```bash
vyn check program.vyn
```

Useful for catching type errors quickly without executing code.

### `disasm` - Disassemble Bytecode

View the compiled bytecode of a program:

```bash
vyn disasm program.vync
```

This shows the low-level bytecode instructions generated by the compiler.

### `version` - Show Version

Display Vyn version information:

```bash
vyn version
```

### `help` - Get Help

Show help information:

```bash
vyn help
vyn help run
```

## Options

### `--no-progress`

Disable the progress bar during compilation:

```bash
vyn run --no-progress program.vyn
```

### `--verbose`

Show detailed timing information for each compilation phase:

```bash
vyn run --verbose program.vyn
```

### `-q, --quiet`

Minimal output mode:

```bash
vyn run --quiet program.vyn
```

### `--slow-mode`

Slow down compilation for debugging purposes:

```bash
vyn run --slow-mode program.vyn
```

### `--time`

Show time taken for each compilation phase:

```bash
vyn run --time program.vyn
```

## Examples

```bash
# Run a program with timing info
vyn run --time hello.vyn

# Check types verbosely
vyn check --verbose program.vyn

# Disassemble with minimal output
vyn disasm --quiet program.vync
```
